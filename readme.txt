This plugin looks up congressional legislators for a given address.

== Description ==
This plugin looks up congressional legislators for a given address using
the google civic api.  You are required to have a google civic api key to
utilize this plugin.

== Frequently Asked Questions ==
1. What service does this use for lookups?
Google Civic API
